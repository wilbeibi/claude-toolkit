# Toolkit Help

This command lists the tools and skills available in this toolkit.

## Available Skills
- **obsidian-rest-api**: Interact with Obsidian vault via Local REST API.
- **dayflow-review**: Query and analyze Dayflow time tracking database.
- **m-cli**: Control macOS system functions from the terminal.

## Usage
You can invoke these skills using natural language. For example:
- "Search my Obsidian notes for..."
- "Summarize my productivity from today in Dayflow."
- "Toggle dark mode using m-cli."
